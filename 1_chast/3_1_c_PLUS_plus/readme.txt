Т.к. в задании сказано, что дан простой одномерный массив, не векторный, не динамический, а одномерный массив, 
состоящий из 15-ти элементов, так что лучше в данной задачи не использовать вектор.

Соответственно, достаточно в программе заменить строки:

 int n;
 n = 15; 
 vector <int> mas(n);

На: 

const int n = 15;
    int mas[n];


Если всё таки в задании необходимо именно переставить элты массива, то ученик уже должен будет сам хотябы попробовать
написать программу для этого, если же ученик не сможет, то мне не составит труда помочь ему с этим.
